# RobenGate Sentinel — Presentación Profesional Interactiva

Presentación de **12 diapositivas** en HTML/CSS/JS para la Práctica Profesional Voluntariado de Robenson Louissaint en el Instituto Profesional IPP — Julio 2026.

---

## 🚀 Cómo Abrir

1. Descarga o clona el repositorio.
2. Navega a la carpeta `presentation/`.
3. Abre el archivo `index.html` directamente en cualquier navegador moderno (Chrome, Firefox, Edge, Safari).

```bash
# Opción rápida con Python (servidor local)
cd presentation
python3 -m http.server 8080
# Luego abre: http://localhost:8080
```

> ℹ️ No se requieren dependencias externas ni compilación. La presentación funciona completamente offline (salvo la carga de fuentes de Google Fonts).

## 🎮 Navegación

| Acción | Tecla / Botón |
|--------|--------------|
| Siguiente diapositiva | `→` / `↓` / `PageDown` / Botón **Siguiente** |
| Diapositiva anterior | `←` / `↑` / `PageUp` / Botón **Anterior** |
| Ir a una diapositiva | Click en los **puntos indicadores** en la barra inferior |
| Pantalla completa | Tecla `F` / Botón **⛶ Pantalla Completa** |
| Salir de pantalla completa | `Esc` |
| Navegación táctil | Deslizar horizontalmente (swipe) |

---

## 📄 Exportar a PDF

### Método 1 — Botón integrado
Haz click en el botón **⬇ Exportar PDF** en la barra de navegación inferior. Esto prepara todas las diapositivas y abre el diálogo de impresión del navegador.

### Método 2 — Ctrl+P / Cmd+P
Presiona `Ctrl+P` (Windows/Linux) o `Cmd+P` (macOS) para abrir el diálogo de impresión.

### Configuración recomendada para PDF
- **Orientación:** Horizontal (Landscape)
- **Tamaño de papel:** A4 o Letter
- **Márgenes:** Ninguno (None)
- **Escala:** Ajustar a la página
- **Destino:** Guardar como PDF

## 🎨 Diseño

- **Tema:** Dark Mode corporativo (Tech/Ciberseguridad)
- **Colores:** `#0A1929` fondo · `#00B4D8` acentos cian · `#FFFFFF` texto
